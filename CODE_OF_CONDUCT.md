# Code of Conduct

This project is an unofficial fan-made World Cup viewing tool. The goal is to keep contributions useful, respectful, and focused on improving the product.

## Expected Behavior

- Be respectful and constructive in issues, discussions, and pull requests.
- Keep feedback specific, actionable, and tied to user experience, data quality, accessibility, or engineering reliability.
- Assume good intent, but accept corrections when a maintainer points out legal, data, or safety constraints.
- Respect third-party rights. Do not submit FIFA logos, mascots, posters, protected marks, or copyrighted media without permission.

## Unacceptable Behavior

- Harassment, insults, personal attacks, or discriminatory language.
- Spam, deceptive promotion, or unrelated self-promotion.
- Attempts to add malicious code, hidden tracking, credentials, or undisclosed network calls.
- Repeatedly pushing data or branding changes that conflict with the project’s unofficial status.

## Enforcement

Maintainers may edit, close, hide, or remove issues, comments, pull requests, or contributions that violate this code of conduct. Serious or repeated violations may result in a block from the repository.

## Reporting

Report concerns privately through GitHub private vulnerability reporting when security-related, or through the repository owner's public GitHub profile for conduct issues.
