# Roadmap

This roadmap focuses on making the project more useful during the 2026 World Cup while keeping it safe to publish as an unofficial fan project.

## v0.1.0 Launch

- Chinese match schedule, city pages, team pages, bracket, and spotlight views.
- Rule-based viewing concierge.
- Local favorites, reminders, single-match and filtered `.ics` calendar export.
- Shareable schedule filter URLs.
- Data status page and 6-hour refresh workflow.

## Near-Term Improvements

- Deploy a public demo and link it from GitHub About and the README.
- Add a PWA manifest and installable mobile shortcut.
- Add English README and UI language toggle.
- Add share cards for individual matches and city schedules.
- Add better parser snapshot tests for upstream page structure changes.
- Add visible data-diff notes after each refresh.

## Good Fit Contributions

- Accessibility checks and keyboard navigation improvements.
- City travel note improvements.
- Schedule parser tests based on saved public HTML snapshots.
- Better copy for international users.
- Additional safe, self-owned visual assets.

## Not Planned

- Official FIFA branding.
- Minute-by-minute live score tracking.
- Login, accounts, or cross-device sync.
- AI or LLM API integration for the rule-based concierge.
