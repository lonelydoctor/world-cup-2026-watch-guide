# v0.1.0 - Launch Release

Initial public release of the unofficial 2026 World Cup watch guide.

## Highlights

- Chinese matchday homepage with next matches and local-time display.
- Full schedule with filters for time, city, country, round, and status.
- Shareable schedule filter URLs.
- Single-match, filtered, and all-match `.ics` calendar export.
- Bracket, city, team, and semifinal/final spotlight pages.
- Rule-based viewing concierge for natural-language-style questions.
- Data status page and `/api/status`.
- Lightweight PWA manifest for mobile install surfaces after deployment.
- GitHub CI, Dependabot, issue templates, release-ready README, security policy, and contribution guidelines.

## Data

- Snapshot generated at `2026-06-22T05:28:16.828Z`.
- 104 total matches.
- 40 finished matches.
- 64 scheduled matches.
- Public schedule source: https://en.wikipedia.org/wiki/2026_FIFA_World_Cup

## Scope and Limitations

- Unofficial fan-made project.
- Not affiliated with, endorsed by, or sponsored by FIFA.
- Does not use FIFA official logos, mascots, posters, or protected marks.
- Not a live minute-by-minute score service.
- Data refresh is designed for periodic post-match updates.
